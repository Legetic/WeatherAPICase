# WeatherAPICase
## Running the program
Below are ways to install and run the program.

### Using an IDE
1. Clone repo to directory of choice
2. Open repo in IDE of choice
3. Run Main.java in the IDE.

### No IDE
1. Clone repo to directory of choice
2. Go to: "WeatherCase\lib\" in a terminal
3. run "java -jar WeatherCase.jar"
(This presumes a JDK version of 1.8 or above installed on the machine and linked to in the PATH variable)
